Este script corresponde al tercer trabajo práctico de la materia Laboratorio de Datos dictada por la UBA para la carrera Lic. en Ciencias de Datos.
El trabajo consiste en utilizar el algoritmo K-Means de clusterización para clasificar imágenes de Pókemons de 56x56 pixeles y en escala de grises.
